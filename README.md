# Karisma Academy - Flutter

Project repository untuk menunjang proses belajar mengajar. Struktur project dibagi menjadi 2 yaitu folder Materi dan folder Chat App.
Pastikan Command Promt atau Terminal Anda berada pada folder project ini.

## Materi

Beberapa potongan code yang ada di slide power point ada di dalam folder ini sesuai sesi.

### Menjalankan folder di salam sesi

Di dalam folder materi bisa langsung di preview atau dijalankan. Terdapat 2 tipe yaitu dart dan flutter

#### Dart

Sesi 1 masih berdiskusi tentang dart. Jadi cara menjalankan code dart dengan format:

`dart run <path-to-dart>`

Contoh kita akan menjalankan file yang ada di dalam sesi-1

```bash
dart run lib/materi/sesi-1/dart_syntax.dart
```

#### Flutter

Di dalam folder materi bisa langsung di preview dengan cara panggil file `main.dart`. Contoh materi sesi 2 android dapat dijalankan dengan script:

```bash
flutter run -t lib/materi/sesi-2/android/main.dart
```
